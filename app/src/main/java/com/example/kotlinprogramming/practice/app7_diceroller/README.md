# 🎲 Dice Roller App

An interactive app that demonstrates state management and user interaction in Jetpack Compose.

## 🚀 What I Learned
- **State Management**: Using `remember` and `mutableStateOf` to handle UI updates.
- **Interactive UI**: Handling button clicks to trigger logic.
- **Conditional Logic**: Changing the dice image dynamically based on a random number.
- **Resources**: Using `painterResource` to load different images at runtime.

## 🛠 Features
- Randomized dice rolling logic.
- Instant UI updates without activity recreation.
- Clean and responsive Material 3 design.
