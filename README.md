# 🚀 Kotlin & Android Learning Journey

A comprehensive repository showcasing my progress in learning **Kotlin Fundamentals** and **Modern Android Development** with **Jetpack Compose**. This project serves as a centralized hub for various practice applications and coding exercises.

---

## 📂 Project Structure

The project is divided into two main modules:

### 📱 1. `:app` (Android Application)
A multi-screen Jetpack Compose application that acts as a launcher for various practice apps:
- **Happy Birthday App**: Basics of Layouts, Text, and Images.
- **Compose Article**: Working with multiple Composables and structured layouts.
- **Dice Roller App**: Learning state management and interactive UIs.
- **Business Card App**: UI design and component styling.

### 💻 2. `:lib` (Kotlin JVM Library)
Dedicated to mastering Kotlin's syntax and logic through real-world scenarios:
- **Kotlin Fundamentals**: Exercises covering Conditionals, Loops, Classes, and Collections.
- **OOP Practice**: In-depth implementation of Object-Oriented Programming concepts.

---

## 🛠 Tech Stack
- **Language**: Kotlin
- **UI Framework**: Jetpack Compose
- **Build System**: Gradle (Kotlin DSL)
- **Architecture**: Single Activity, State-driven Navigation

---

## 📖 How to Run
1. Clone this repository.
2. Open in **Android Studio (Ladybug or newer)**.
3. Sync Gradle.
4. Run the `:app` module on an Emulator or Physical Device.
5. To run Kotlin exercises, navigate to the `:lib` module and run the `main()` functions.
