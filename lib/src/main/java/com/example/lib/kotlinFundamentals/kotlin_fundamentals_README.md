# 💻 Kotlin Fundamentals & OOP Practice

This module contains various Kotlin exercises and Object-Oriented Programming (OOP) implementations. It serves as the logical backbone of my learning journey.

## 🧠 Concepts Covered
- **Basic Syntax**: Variables, Nullability, and Conditionals.
- **Functional Programming**: Lambda expressions and Higher-order functions.
- **Object-Oriented Programming**:
    - Classes and Objects
    - Inheritance & Polymorphism
    - Interfaces & Abstract Classes
- **Real-world Scenarios**:
    - `songCatalog.kt`: Working with collections and data classes.
    - `movieTicketPrice.kt`: Implementing business logic with `when` expressions.
    - `InternetProfile.kt`: Handling structured data and printing formatted output.

## 🛠 How to Run
Every file contains a `main()` function. You can run them directly within Android Studio to see the output in the console.
